# Def

Application written in Golang to allow for the user to define a given term.

## Usage

`def [-l language\_code] word`

## Language Codes
| Code | Language |
| ---- | -------- |
| ars  | Arabic |
| des  | German |
| ens  | English |
| ess  | Spanish |
| frs  | French |
| his  | Hindi |
| its  | Italian |
| jas  | Japanese |
| kos  | Korean |
| pt-BR| Brazilian Portuguese |
| ru   | Russian |
| tr   | Turkish
| zh-CN| Chinese (Simplified) |
